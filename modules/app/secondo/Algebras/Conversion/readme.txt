For using this converter, the tool unrtf is required. To install it:

SuSe Linux
==========

download it from:
http://www.gnu.org/software/unrtf/unrtf-0.21.9.tar.gz
unpack the files into some directory
open a terminal and navigate into that directory
call the following linux commands...

./configure
make
sudo make install

Ubuntu
======

sudo apt-get install unrtf
